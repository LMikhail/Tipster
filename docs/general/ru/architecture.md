2.  **–õ–µ–Ω–∏–≤—ã–µ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏:** –í–æ–∑–≤—Ä–∞—Ç —Ä–µ–∑—É–ª—å—Ç–∞—Ç–æ–≤ –≤ –≤–∏–¥–µ –ª–µ–Ω–∏–≤—ã—Ö (lazy) –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–µ–π ‚Äî —ç—Ç–æ –∏–¥–∏–æ–º–∞—Ç–∏—á–Ω–æ–µ –¥–ª—è Clojure —Ä–µ—à–µ–Ω–∏–µ. –û–Ω–æ –∫—Ä–∞–π–Ω–µ —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ –ø–æ –ø–∞–º—è—Ç–∏, —Ç–∞–∫ –∫–∞–∫ –ø–æ–∑–≤–æ–ª—è–µ—Ç –æ–±—Ä–∞–±–∞—Ç—ã–≤–∞—Ç—å –ø–æ—Ç–µ–Ω—Ü–∏–∞–ª—å–Ω–æ –±–µ—Å–∫–æ–Ω–µ—á–Ω—ã–µ –∏–ª–∏ –æ—á–µ–Ω—å –±–æ–ª—å—à–∏–µ –Ω–∞–±–æ—Ä—ã —Ä–µ—à–µ–Ω–∏–π, –Ω–µ –∑–∞–≥—Ä—É–∂–∞—è –∏—Ö –≤—Å–µ –≤ –ø–∞–º—è—Ç—å —Å—Ä–∞–∑—É.

### Intelligent Pipeline

**–û—Ç–≤–µ—á–∞–µ—Ç –∑–∞:** –°–æ–∑–¥–∞–Ω–∏–µ –æ–ø—Ç–∏–º–∞–ª—å–Ω–æ–π —Å—Ç—Ä–∞—Ç–µ–≥–∏–∏ (–ø–ª–∞–Ω–∞) –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è –¥–ª—è —Å–ª–æ–∂–Ω—ã—Ö –∑–∞–ø—Ä–æ—Å–æ–≤. –ï—Å–ª–∏ –î–≤–∏–∂–æ–∫ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è ‚Äî —ç—Ç–æ "–º—É—Å–∫—É–ª—ã", —Ç–æ Intelligent Pipeline ‚Äî —ç—Ç–æ "–º–æ–∑–≥", –∫–æ—Ç–æ—Ä—ã–π —Ä–µ—à–∞–µ—Ç, –∫–∞–∫ —ç—Ç–∏–º–∏ –º—É—Å–∫—É–ª–∞–º–∏ –ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è –Ω–∞–∏–±–æ–ª–µ–µ —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ.

*   **–í—Ö–æ–¥:** –°–ª–æ–∂–Ω–∞—è –ª–æ–≥–∏—á–µ—Å–∫–∞—è —Ü–µ–ª—å –æ—Ç [–î–≤–∏–∂–∫–∞ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è](#–¥–≤–∏–∂–æ–∫-–≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è).

*   **–í—ã—Ö–æ–¥:** –û–ø—Ç–∏–º–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω—ã–π –ø–ª–∞–Ω –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è, –∫–æ—Ç–æ—Ä—ã–π –ø–µ—Ä–µ–¥–∞–µ—Ç—Å—è –æ–±—Ä–∞—Ç–Ω–æ [–î–≤–∏–∂–∫—É –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è](#–¥–≤–∏–∂–æ–∫-–≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è).

**–ê—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–Ω–æ–µ —Ä–µ—à–µ–Ω–∏–µ ("–ü–æ—á–µ–º—É"):**
–≠—Ç–æ –Ω–µ –º–æ–Ω–æ–ª–∏—Ç–Ω—ã–π –∫–æ–º–ø–æ–Ω–µ–Ω—Ç, –∞ –∫–æ–Ω–≤–µ–π–µ—Ä –∏–∑ –¥–≤—É—Ö –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤—É—é—â–∏—Ö —á–∞—Å—Ç–µ–π. –¢–∞–∫–æ–µ —Ä–∞–∑–¥–µ–ª–µ–Ω–∏–µ –ø–æ–∑–≤–æ–ª—è–µ—Ç —Å–æ—á–µ—Ç–∞—Ç—å –º–æ—â—å –¥–∏–Ω–∞–º–∏—á–µ—Å–∫–æ–≥–æ –∞–Ω–∞–ª–∏–∑–∞ —Å –ø—Ä–æ–∏–∑–≤–æ–¥–∏—Ç–µ–ª—å–Ω–æ—Å—Ç—å—é —Å–∫–æ–º–ø–∏–ª–∏—Ä–æ–≤–∞–Ω–Ω–æ–≥–æ –∫–æ–¥–∞.

### API –°–ª–æ–π –∏ –ê–¥–∞–ø—Ç–µ—Ä—ã
